You will need to compile this example first:
  mvn compile

To run the example type
  mvn camel:run

To stop the example hit ctrl + c

To generate javadoc, testreport and code-coverage run
	mvn site
	
Javadoc: 		target/site/apidocs/index.html
Testreport: 	target/site/testreport/surefire-report.html
Code-Coverage:	target/site/cobertura/index.html
